Spielanleitungen

Taktisches Puzzle

In diesem Spiel musst du Puzzlestücke bewegen, um das Puzzle zu lösen:

  1. Puzzlestück auswählen: Klicke auf ein Puzzlestück.
  2. Plätze tauschen: Klicke auf ein anderes Puzzlestück, um die Positionen zu wechseln.

Sobald das Puzzle gelöst ist, erscheint ein informativer Text über eine historische Persönlichkeit.

Sofortige Lösung: Klicke auf „Puzzle lösen“, um das Puzzle sofort zu lösen und die Informationen anzuzeigen.

Marie Curie's Game

In diesem Spiel wählst du Elemente aus dem Periodensystem aus:

  1. Elemente auswählen: Klicke auf die Elemente, die du für korrekt hältst.
  2. Rückmeldung: Wenn du die richtigen Elemente auswählst, erscheint Marie Curie und bestätigt, dass deine Auswahl korrekt ist.
  3. Anzahl der Elemente: Du kannst zwischen 1 und 6 Elemente auswählen.
  4. Puzzle lösen: Drücke auf „Solve“, um deine Auswahl zu überprüfen.

Code Cracker

In diesem Spiel gilt es, den Java-Code zu entschlüsseln:

  1. Code anzeigen: Auf dem Bildschirm erscheint ein Java-Code.
  2. Eingabe: XGib in dem Eingabefeld darunter ein, was die Ausgabe des Codes ist oder was der Code macht.
  3. Bestätigen: Klicke auf „Überprüfen“, um zu erfahren, ob deine Antwort richtig ist oder nicht.
